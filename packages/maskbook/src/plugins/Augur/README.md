# Plugin: Augur Turbo

This plugin fetch and show [Augur](https://www.augur.net/) prediction markets allow user to predict on them.

## Feature Set

- [] Detect Augur URL
- [] Show market info
- [] Draw market chart
- [] Buy/Sell
