# Plugin: Augur Turbo

This plugin fetch and show [Augur](https://www.augur.net/) prediction markets allow user to predict on them.

## Feature Set

- [x] Detect Augur URL
- [x] Show market info
- [x] Buy/Sell
- [x] Draw market chart
- [x] Add/Remove Liquidity
- [ ] NFL Market
- [ ] Crypto Market
- [ ] Portfolio
