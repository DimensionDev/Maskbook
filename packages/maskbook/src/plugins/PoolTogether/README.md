# Plugin: PoolTogether

This plugin fetch and show [PoolTogether](https://pooltogether.com) pools data and allow user to invest on that poll and see her/his account data.

## Feature Set

- [x] Detect PoolTogether address
- [x] Show pools list
- [x] Deposit
- [x] My account tab
