# Plugin: dHEDGE

This plugin fetch and show [dHEDGE](https://dhedge.org) pools data and allow user to invest on that poll.

## Feature Set

- [x] Detect dHEDGE pool
- [x] Show pool info
- [x] Draw pool chart
- [x] Buy sUSD
- [x] Investment
- [x] Multi Chain
- [x] V2 Pools
