# Plugin: Reality Cards

This plugin fetch and show [Reality Cards](https://realitycards.io) markets and allow user to rent/sell a card.

## Feature Set

- [x] Detect RealityCards market url
- [x] Show market info and cards
- [x] Rent a card
- [x] Deposit
- [x] Exit position
- [x] Increase rent
